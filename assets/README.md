# Assets

This folder contains app assets including:

- `icon.png` - App icon (1024x1024)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `splash.png` - Splash screen image (1242x2688)
- `favicon.png` - Web favicon (48x48)

## Design Guidelines

All assets should follow the BoomGhoom brand guidelines:

### Logo Colors (Gradient)
- Orange: `#FF8A50`
- Magenta: `#E066A0`
- Purple: `#9B6DFF`
- Blue: `#5B8DEF`

### Background Colors
- Dark: `#0A0A0F`
- Light: `#FAFAFA`

## Creating Assets

Use the logo provided in the design specs. The logo consists of:
- A gradient bowl/orbit shape
- A gradient circle positioned above the bowl
- Curly elements on both sides

For the splash screen, place the logo centered on a dark gradient background.
